# What is Android?
Android is an Operating System based on [Linux](https://en.wikipedia.org/wiki/Linux), it is used by all phones except iPhone-like devices.

## Ressemblances to Linux
Like Linux, Android has multiple flavours (distros) like [LineageOS](https://en.wikipedia.org/wiki/LineageOS) (commonly used troughout this documentation). \
Android is also **open-source**, it's code is available [here](https://android.googlesource.com/).

## Dfferences to Linux
Unlike Linux, it isn't meant to be a **desktop** operating system, even tho there were some attempts like **Samsung Dex**.
