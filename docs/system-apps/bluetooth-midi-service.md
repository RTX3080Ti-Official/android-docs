# Bluetooth MIDI Service
This app the **core** of the MIDI Service, it's package name is ``com.android.bluetoothmidiservice``, it might be a depedency of [Bluetooth](./bluetooth.md)

:::warning

Disbaling this app will remove the ability to connect to a MIDI device via Bluetooth.

:::

## Services
This app only has one service named **BluetoothMidiService**.
