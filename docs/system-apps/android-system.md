# Android System
This app is one of the **core apps** of Android. It has the Android version as it's version. The package name is just ``android``.

## Activities
**Android System** mainly has dialogs activties all apps can launch, for example: **Choose an Account** (``.accounts.ChooseAccountTypeActivity``)

## Services
**Android System** has many services having a ``com.android.*`` package name, only 2 of them are under the same package as the app.

## Receivers
**Android System** has receivers under the ``com.android.*`` package name.

## Providers
**Android System** only has one provider, named **Icons Content Provider** under the ``com.android.*``

## Conclusion
This app is a core app of Android, which has almost all the dialogs used in apps.

:::danger

Do not try disabling or deleting **Android System**, this may cause the system to become unstable or not even boot.

:::
