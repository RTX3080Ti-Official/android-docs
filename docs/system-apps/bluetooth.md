# Bluetooth
**Bluetooth**, with it's package name ``com.android.bluetooth`` is an essential part of the Bluetooth system on Android systems.

:::warning

Deleting this app will totally break the Bluetooth functionnality of the system.

:::

## Functions
This app handles all the dialogs and activties for **Bluetooth file transfers**, **Connection errors**, etc.

It also stores all it's config in the **SharedPrefs**.

:::tip

If you have a rooted device and wanna quickly clear out all permissions assigned to Bluetooth devices, empty the ``message_access_permission.xml``, ``phonebook_access_permission.xml`` and ``sim_access_permission.xml`` files.

:::
