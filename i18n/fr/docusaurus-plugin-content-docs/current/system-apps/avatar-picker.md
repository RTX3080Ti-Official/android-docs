# Avatar Picker

**Avatar Picker** is an app that lets you assign pictures to contacts. It's package name is `com.android.avatarpicker`.

## Activities

Not counting all the **AndroidX** activities, there is only one named **Avatar Picker Activity**.

This activity lets you assign a picture by selecting a default one, taking a picture or selecting one from Internal Storage.

:::warning

This app might be a depedency of the **Contacts** app.

:::
