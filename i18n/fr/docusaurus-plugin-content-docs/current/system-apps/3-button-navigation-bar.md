# Barre de navigation à 3 boutons

Cette appli semble être une **application factice**. Il s'agit probablement de la barre de navigation par défaut du système lorsque vous n'utilisez pas de commandes de geste. Son nom de package est `com.android.internal.systemui.navbar.threebutton`.

:::warning

Désactiver ou supprimer ce fichier **peut** rendre inutilisable la barre de navigation à 3 boutons.

:::
