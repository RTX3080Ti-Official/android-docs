# Captive Portal Login

**Captive Portal Login**, with the package name `com.android.captivelogin`, as it names says, allows you to connect to Captive Portal WiFi (e.g. hotel WiFi).

:::note

On ROMs with **Google Play Services**, this app package name is `com.google.android.captiveportallogin` and it's signature is Google's.

:::

## Functions

This app is probably a simple WebView activity, we can confirm this beacause there is a service named **DownloadService**.

## Activities

This app only has one activity named **Sign in to network**.
